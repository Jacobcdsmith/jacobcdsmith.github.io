```css
:root {
    --color-primary: #00D9FF;      /* Electric Blue */
    --color-secondary: #FF0055;    /* Crimson Red */
```

Adjust particle system in `script.js`:

```javascript
this.particleCount = 80;           // Number of particles
this.mouse.radius = 150;           // Mouse interaction radius
```
